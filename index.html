How grading works

Official fertility definition

Step 01Build wiki-faithful Markdown (keep URLs, punct, tables).
Step 02Faithful unit = letter/mark/number run OR one visible symbol.
Step 03X = BPE tokens ÷ faithful units (must be ≤ 1.2).
Step 04score = 1000 / (Xmax − Xmin); Hindi penalty if HI > 1.2.
Formulas faithful_unit = [\p{L}\p{M}\p{N}]+ | [^\s\p{L}\p{M}\p{N}] fertility(lang) = token_count / faithful_unit_count score = 1000 / (max_fertility − min_fertility) hindi_penalty = exp(max(0, hindi_fertility/1.2 − 1)) hindi_adjusted = score / hindi_penalty
